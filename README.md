# object-pick-ts


## Install
Install with npm:

```$ npm install --save object-pick-ts```

## Usage

```ts
import {ObjectPick as pick} from "object-pick-ts";
 
let result = pick({a: 'a', b: 'b', c: 2}, ['a', 'c']);

//{a: 'a', c: 2};
```
